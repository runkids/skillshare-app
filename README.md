## Prerequisites

Install [skillshare CLI](https://github.com/runkids/skillshare) first.

## Install

### Homebrew (macOS)

```bash
brew tap runkids/tap
brew install --cask skillshare-app

# Uninstall
brew uninstall --cask skillshare-app
```
