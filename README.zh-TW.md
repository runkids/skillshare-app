## 📍 為什麼選擇 Skillshare App？

厭倦了手動管理散落各處的 dotfiles 和設定檔？

Skillshare App 讓你的 dotfiles 管理變得直覺又高效 — 一個 App 搞定設定同步、版本管理、自動化操作。使用 **Tauri + Rust** 打造，安裝檔超小、啟動超快。

專為 **vibe coding** 設計：少點手動搬移設定的混亂，多點時間寫酷東西。也更重要的是：你可以把日常操作交給 AI（透過 MCP）來幫你做。

## 🎬 快速開始

1. **下載** → [下載最新版本](https://github.com/runkids/skillshare-app/releases)
2. **開啟專案** → 拖曳任何含設定檔的資料夾到 App
3. **開始使用** → 點擊任何操作卡片即可執行！

> 💡 **小技巧**：按 <kbd>Cmd</kbd>+<kbd>K</kbd> 快速切換 worktrees

## ✨ 功能特色

| | 功能 | 說明 |
|:---:|---------|-------------|
| 🎯 | **一鍵執行命令** | 卡片式執行常用指令，即時 PTY 終端輸出 |

## 🤖 AI & MCP

Skillshare App 的「AI 驅動」不是只會聊天，而是 AI 可以呼叫真正的工具來完成工作（MCP）。

**AI 實際會做什麼（MCP Tool Chain 範例）**
- 「列出我的專案，並顯示某個專案的設定」→ `list_projects` → `get_project`
- 「幫我跑 `test` 並摘要失敗原因」→ `get_project` → `run_npm_script`
- 「根據 staged changes 產生 Conventional Commit 訊息」→ `get_git_diff` →（AI 生成 commit message）

**快速設定**
- 在 Skillshare App 打開 **Settings → MCP → MCP Integration**，複製產生的設定（JSON/TOML）到你的 AI 工具（Claude Code / Codex CLI 等）
- MCP server 使用 stdio，不需要開 port

## 📦 安裝

### Homebrew (macOS)

```bash
brew tap runkids/tap
brew install --cask skillshare-app
```

#### 升級

```bash
brew update && brew upgrade --cask skillshare-app
```

#### 疑難排解

如果升級時出現 `It seems the App source '/Applications/Skillshare App.app' is not there` 錯誤：

```bash
# 強制卸載舊的 cask 記錄
brew uninstall --cask --force skillshare-app

# 重新安裝
brew install --cask runkids/tap/skillshare-app
```

### 手動下載

從 [Releases](https://github.com/runkids/skillshare-app/releases) 頁面下載最新版本。

## 🛠 開發

### 前置需求

- Node.js 18+
- Rust 1.70+
- pnpm

### 設定

```bash
# Clone 專案
git clone https://github.com/runkids/skillshare-app.git
cd skillshare-app

# 安裝依賴
pnpm install

# 啟動 Vite (web UI)
pnpm dev

# 啟動桌面應用
pnpm dev:tauri
```

### 建置

```bash
# 建置 web 資源
pnpm build

# 建置桌面應用 (dmg)
pnpm build:tauri
```
