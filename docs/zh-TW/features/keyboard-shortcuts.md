# 鍵盤快捷鍵

Skillshare App 所有鍵盤快捷鍵的完整參考。

## 概覽

Skillshare App 專為鍵盤優先的工作流程設計。學習這些快捷鍵以更快導覽。

## 快速參考

### 全域快捷鍵

| 快捷鍵 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>K</kbd> | 快速切換 worktree |
| <kbd>Cmd</kbd> + <kbd>1</kbd> | 前往專案分頁 |
| <kbd>Cmd</kbd> + <kbd>2</kbd> | 前往工作流程分頁 |
| <kbd>Cmd</kbd> + <kbd>,</kbd> | 開啟設定 |
| <kbd>Cmd</kbd> + <kbd>/</kbd> | 顯示所有快捷鍵 |
| <kbd>Cmd</kbd> + <kbd>R</kbd> | 重新整理目前檢視 |
| <kbd>Cmd</kbd> + <kbd>N</kbd> | 新增項目（依上下文） |
| <kbd>Cmd</kbd> + <kbd>S</kbd> | 儲存目前項目 |
| <kbd>Cmd</kbd> + <kbd>W</kbd> | 關閉目前面板 |
| <kbd>Escape</kbd> | 關閉對話框 / 取消 |

| <kbd>Cmd</kbd> + <kbd>↑</kbd> | 上一個專案 |

| <kbd>Tab</kbd> | 下一個可聚焦元素 |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | 上一個可聚焦元素 |

### 專案面板

| <kbd>Cmd</kbd> + <kbd>F</kbd> | 搜尋專案 |
| <kbd>Delete</kbd> | 移除選中的專案 |

| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd> | 取消暫存所有變更 |

| <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> | Pull 變更 |

### 工作流程編輯器

|--------|------|

| <kbd>Cmd</kbd> + <kbd>C</kbd> | 複製選中的節點 |

| <kbd>Delete</kbd> | 刪除選中的節點 |

### 終端機

| 快捷鍵 | 操作 |
|--------|------|
| <kbd>Cmd</kbd> + <kbd>C</kbd> | 複製（選中文字時） |
| <kbd>Cmd</kbd> + <kbd>K</kbd> | 清除終端機 |
| <kbd>Ctrl</kbd> + <kbd>C</kbd> | 停止執行中的腳本 |

## 自訂快捷鍵

### 編輯快捷鍵

1. 前往**設定** → **鍵盤快捷鍵**
2. 找到要變更的快捷鍵
3. 點擊目前的按鍵組合
4. 按下您的新按鍵組合
5. 點擊**儲存**

<!-- TODO: Add screenshot of shortcut editor -->

### 重設為預設值

1. 前往**設定** → **鍵盤快捷鍵**
2. 點擊**全部重設為預設值**
3. 確認重設

### 衝突偵測

Skillshare App 會在以下情況警告您：
- 快捷鍵與另一個衝突
- 快捷鍵可能與系統快捷鍵衝突

## 快捷鍵分類

### 導覽快捷鍵

在應用程式中快速移動：

- 切換分頁
- 導覽專案列表
- 開啟/關閉面板

### 操作快捷鍵

執行常見操作：

- 執行腳本
- 提交變更
- 部署專案

### 編輯器快捷鍵

標準編輯快捷鍵：

- 復原/重做
- 複製/貼上
- 全選

## 進階使用者提示

### 逐步學習

1. 從最常用的開始：<kbd>Cmd</kbd> + <kbd>K</kbd>
2. 每天新增一個快捷鍵
3. 練習直到成為肌肉記憶

### 使用浮動按鈕

浮動快捷鍵按鈕（右下角）顯示上下文相關的快捷鍵。

### 快速切換最重要

<kbd>Cmd</kbd> + <kbd>K</kbd> 是最強大的快捷鍵：
- 從任何地方都可使用
- 輸入時即時過濾
- 優先顯示最近的 worktree

## 無障礙

### 焦點管理

Skillshare App 為鍵盤導覽維護正確的焦點順序。

### 螢幕閱讀器支援

所有互動元素都有適當的標籤供螢幕閱讀器使用。

### 減少動態效果

啟用減少動態效果設定時，快捷鍵運作方式相同。

## 平台說明

### macOS

所有快捷鍵使用 <kbd>Cmd</kbd> 作為修飾鍵。

### Windows/Linux（即將推出）

快捷鍵將使用 <kbd>Ctrl</kbd> 代替 <kbd>Cmd</kbd>。

## 疑難排解

### 快捷鍵無效

1. 檢查是否有對話框開啟（按 <kbd>Escape</kbd>）
2. 在設定中驗證快捷鍵
3. 檢查與其他應用程式的衝突
4. 嘗試重設為預設值

### 與系統快捷鍵衝突

某些系統快捷鍵優先：
- <kbd>Cmd</kbd> + <kbd>Space</kbd>（Spotlight）
- <kbd>Cmd</kbd> + <kbd>Tab</kbd>（應用程式切換器）

Skillshare App 無法覆寫這些。

### 自訂快捷鍵未儲存

1. 檢查衝突
2. 確保組合有效
3. 嘗試不同的組合
