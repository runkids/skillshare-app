## 📍 为什么选择 Skillshare App？

厌倦了手动管理散落各处的 dotfiles 和配置文件？

Skillshare App 让你的 dotfiles 管理变得直觉又高效 — 一个 App 搞定配置同步、版本管理、自动化操作。使用 **Tauri + Rust** 打造，安装包超小、启动超快。

专为 **vibe coding** 设计：少点手动搬移配置的混乱，多点时间写酷东西。更重要的是：你可以把日常操作交给 AI（通过 MCP）来帮你做。

## 🎬 快速开始

1. **下载** → [下载最新版本](https://github.com/runkids/skillshare-app/releases)
2. **打开项目** → 拖拽任何含配置文件的文件夹到 App
3. **开始使用** → 点击任何操作卡片即可执行！

> 💡 **小技巧**：按 <kbd>Cmd</kbd>+<kbd>K</kbd> 快速切换 worktrees

## 🤖 AI & MCP

Skillshare App 的「AI 驱动」不是只会聊天，而是 AI 可以调用真正的工具来完成工作（MCP）。

**AI 实际会做什么（MCP Tool Chain 示例）**
- 「列出我的项目，并显示某个项目的配置」→ `list_projects` → `get_project`
- 「帮我跑 `test` 并总结失败原因」→ `get_project` → `run_npm_script`
- 「根据 staged changes 生成 Conventional Commit 信息」→ `get_git_diff` →（AI 生成 commit message）

**快速设置**
- 在 Skillshare App 打开 **Settings → MCP → MCP Integration**，复制生成的配置（JSON/TOML）到你的 AI 工具（Claude Code / Codex CLI 等）
- MCP server 使用 stdio，不需要开 port

## 📦 安装

### Homebrew (macOS)

```bash
brew tap runkids/tap
brew install --cask skillshare-app
```

#### 升级

```bash
brew update && brew upgrade --cask skillshare-app
```

#### 故障排查

如果升级时出现 `It seems the App source '/Applications/Skillshare App.app' is not there` 错误：

```bash
# 强制卸载旧的 cask 记录
brew uninstall --cask --force skillshare-app

# 重新安装
brew install --cask runkids/tap/skillshare-app
```

### 手动下载

从 [Releases](https://github.com/runkids/skillshare-app/releases) 页面下载最新版本。

## 🛠 开发

### 前置条件

- Node.js 18+
- Rust 1.70+
- pnpm

### 设置

```bash
# Clone 项目
git clone https://github.com/runkids/skillshare-app.git
cd skillshare-app

# 安装依赖
pnpm install

# 启动 Vite (web UI)
pnpm dev

# 启动桌面应用
pnpm dev:tauri
```

### 构建

```bash
# 构建 web 资源
pnpm build

# 构建桌面应用 (dmg)
pnpm build:tauri
```
